# Propose Love CMS 💖

A beautiful, interactive, and customizable website template for proposing to your loved one.

## Features
- **Dynamic Content via JSON**: No need to touch HTML/CSS to change names, stories, or photos.
- **Visual CMS (Admin Panel)**: An easy-to-use graphical interface at `http://localhost:3000/cms.html` to edit your content.
- **Auto-Commit**: The CMS automatically saves your changes and pushes them to your GitHub repository!
- **Sections Included**: 
  - Hero (Names & Intro)
  - Our Timeline (Your story)
  - Memories Gallery (Photo grid)
  - Why I Love You (Reasons)
  - The Proposal (Interactive Yes/No buttons)

## How to Run

1. Make sure you have **Node.js** installed.
2. Clone this repository.
3. Open terminal/command prompt in the project folder and run:
   ```bash
   node server.js
   ```
4. Open `http://localhost:3000/cms.html` to edit your content.
5. Open `http://localhost:3000/` to view the website.
